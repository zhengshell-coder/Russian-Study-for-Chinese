# 完整标准 33 字母课程规划 v1

## 文档信息
- 版本：`v1.0`
- 日期：`2026-04-18`
- 适用阶段：阶段一《字母与发音基础》

## 1. 课程总目标
- 完整掌握 33 个俄语字母
- 明确区分与拉丁字母相似但不同的俄文字母
- 建立基础音值认知
- 能把字母放进例词中识别和跟读

## 2. 课程结构
本阶段建议拆成 `8` 个单元。

### 单元安排
1. 元音基础 1：`А / О / У / Э`
2. 元音基础 2：`Е / Ё / И / Ы / Ю / Я`
3. 高频相似字母 1：`К / М / Т / А / О`
4. 高频相似字母 2：`В / Н / Р / С / Х`
5. 常见辅音 1：`Б / П / Д / Г / З`
6. 常见辅音 2：`Л / Н / Р / Ф / Ж`
7. 特殊辅音：`Ц / Ч / Ш / Щ / Й`
8. 特殊符号：`Ъ / Ь` + 全字母复习

## 4. 33 字母条目

### 字母
- 字母：`А а`
- 标准名称（俄语）：`а`
- 中文提示：阿
- 转写：`a`
- 发音提示：类似清晰的 `a`
- 例词：`Анна`
- 易混提醒：外形像拉丁 `A`，但学习时按俄文字母处理
