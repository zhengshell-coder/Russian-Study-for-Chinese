# 俄语学习工具评估标准 v1

## 使用规则
- 每完成一轮明显改动，都必须按本标准自评一次
- 必须分维度打分，不能只给总分
- 低于达标线时，必须明确下一轮前 3 项

## 评分方式
- 每个一级维度按 `0-5` 分评分
- 按权重折算为总分 `100`

## 达标线
- 总分至少 `75/100`
- 以下核心维度不能低于 `3/5`
  - 页面显示与交互
  - 实际学习效果
  - 音频学习与跟读
  - 课程设计

## 1. 页面显示与交互体验 `25%`
目标：
- 打开后 5 秒内能开始学习
- 页面清爽
- 当前只聚焦一步

评分点：
1. 是否是单路径学习
2. 是否文案短而清楚
3. 是否像学习页而不是介绍页
4. 是否适合手机竖屏
5. 是否容易继续下一步

## 2. 实际学习效果 `25%`
目标：
- 用户不是“看过”，而是完成最小学习闭环

评分点：
1. 每步目标是否明确
2. 是否控制在 20-30 分钟内
3. 是否同时包含输入和输出
4. 是否有即时反馈
5. 学完后是否有明确收获

## 3. 音频学习与跟读 `25%`
目标：
- 用户能听得清、跟得上、知道哪里还不稳

评分点：
1. 是否支持慢速和正常速
2. 是否能对字母、词、句分别播放
3. 是否有明确跟读入口
4. 是否有基础反馈
5. 样本音质是否足够支撑学习

## 4. 课程设计 `15%`
目标：
- 课程像一条路径，而不是内容拼盘

评分点：
1. 顺序是否自然
2. 内容量是否合理
3. 词汇是否在场景中学习
4. 是否符合中文母语用户理解习惯
5. 是否便于后续扩课

## 5. 技术可用性与手机准备度 `10%`
目标：
- 当前版本简单，但未来不被架构拖住

评分点：
1. 手机浏览器是否可用
2. 数据和界面是否解耦
3. 语音逻辑是否可继续增强
4. 是否具备稳定的 PWA 使用能力
5. 是否避免临时性硬编码
