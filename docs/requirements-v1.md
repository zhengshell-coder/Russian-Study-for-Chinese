# 俄语学习工具需求文档 v1

## 文档信息
- 版本：`v1.0`
- 日期：`2026-04-18`
- 状态：当前有效需求基线
- 适用范围：个人自用、非商业、不对外发布

## 1. 产品定位
一款面向中文母语、可结合英语辅助理解的个人俄语学习工具。产品借鉴多邻国在“课程结构清晰、场景化推进、低决策成本”上的优点，但弱化游戏化，仅保留高频学习、场景学习和基础复习功能。

## 2. 当前范围
当前版本需求只保留以下核心能力：
- 俄语字母学习
- 场景化词汇与句型学习
- 语音播放与基础跟读
- 每日 20-30 分钟学习节奏
- 手机浏览器友好的单题推进体验
- PWA 方式在 iPhone 主屏幕打开使用
- GitHub Pages 上线与 HTTPS 访问

以下需求不再纳入当前范围：
- 手写练习
- 印刷体与手写体区分训练
- 描红、临写、笔顺训练

## 3. 项目目标
- 帮助用户系统学习俄语字母、词汇、句子和场景表达
- 支持每日 20-30 分钟的稳定学习
- 支持字母、单词、句子的语音学习与跟读
- 以多邻国式 Lesson 题流降低学习负担
- 以浏览器可打开的 HTML Demo 起步，并演进到可安装的 PWA 形态

## 4. 核心设计原则
- 轻量优先：先做可持续使用的最小闭环
- 学习优先：不做重游戏化，不做复杂运营机制
- 场景优先：词汇和句型按生活场景组织
- 输出优先：每课至少包含识别、听读、跟读中的两项
- 手机友好：从第一版开始按移动端使用习惯设计
- 字母准确：俄文字母必须独立展示，不能混同为拉丁字母

## 4.1 俄语文本输出规则
- 所有俄语正文一律使用标准俄语西里尔字母输出
- 不允许用外形相似的拉丁字母替代俄文字母
- 转写必须单独放在 `translit` 或“转写”字段中
- 中文说明字段只写中文，不混入俄文字母名称
- 若需要比较俄文字母和拉丁字母，必须明确写出“这是相似字形，不是同一个字母”

## 5. 学习方法要求

### 5.1 方法论
- 高频优先：先学高频字母、高频词、高频场景
- 小步快跑：每天少量新内容，持续推进
- 间隔复习：D1 / D3 / D7 / D14 节奏复习
- 检索练习：通过识别、选择、跟读增强记忆
- 输入输出结合：听、读、说同步推进
- 情境化学习：单词必须进入句子和场景中学习
- 模仿优先：语音学习以听辨、跟读、重音模仿为主
- 场景内复用：同一天的词、句型、听力、抄写、跟读必须互相关联

### 5.2 每日任务节奏
- 每日 Lesson：8-12 道题
- 词汇识别与匹配：2-3 题
- 场景句拼装与听音排序：3-4 题
- 语音跟读：2-3 题
- 错题或快速复盘：1-2 题

## 6. 课程体系

### 6.1 阶段一：字母与发音
目标：
- 认识基础俄语字母
- 区分俄文字母与相似拉丁字母
- 掌握基础发音

内容：
- 字母卡片
- 字母名称
- 转写说明
- 例词
- 发音音频

### 6.2 阶段二：基础词汇
目标：
- 建立高频基础词汇

主题：
- 问候与礼貌
- 自我介绍
- 食物与饮料
- 数字与时间
- 日常活动

### 6.3 阶段三：基础句型
目标：
- 用简单句表达身份、需求、喜好和位置

句型示例：
- 我是……
- 我想要……
- 在哪里？
- 多少钱？

### 6.4 阶段四：场景表达
目标：
- 在微场景中理解和使用表达

场景：
- 打招呼
- 点餐
- 购物
- 问路

## 7. 功能清单

### 7.1 MVP 必须包含
1. 今日 Lesson 题流
2. 题型引擎
3. 单词匹配题
4. 句子拼装题
5. 听音排序题
6. 听句选词题
7. 俄语抄写 / 默写题
8. 语音学习与基础跟读模块
9. 普通速度播放与慢速逐词播放
10. 即时反馈与中文含义解释
11. 进度记录与错题回收
12. PWA 安装能力与 HTTPS 打开能力

### 7.2 字母学习模块
目标：
- 完成认字、发音、转写和例词入门

能力：
- 字母卡片
- 发音播放
- 例词
- 相似拉丁字母提醒

### 7.3 场景模块
目标：
- 从词汇走向表达

能力：
- 场景词包
- 高频句型
- 词汇与句子音频
- 句子可播放并可进入跟读
- 场景句进入拼装题、听音排序题和跟读题

### 7.4 语音模块
目标：
- 支持听、跟读和基础反馈

MVP 范围：
- 慢速 / 正常速度播放
- 跟读入口
- 基础识别反馈
- 按词命中提示
- “现在不方便说”跳过机制

### 7.5 题型引擎
目标：
- 让学习过程从“看卡片”升级为“做题反馈”

MVP 范围：
- 每日题流
- Lesson 进度条
- 检查 / 继续
- 正确 / 错误即时反馈
- 中文 Meaning 解释
- 错题本地记录

## 8. 非功能要求
- 页面加载快
- 结构清晰，5 秒内可开始学习
- 手机竖屏下可舒适操作
- 学习进度本地保存
- 数据结构可扩展，便于后续扩课和优化音频能力
- 支持在 iPhone Safari 中“添加到主屏幕”
- 支持基础离线缓存

## 9. 当前成功标准
- 用户能完成每日 20-30 分钟学习
- 用户能识别首批俄文字母，并区分相似拉丁字母
- 用户能掌握首批场景化高频词汇与句型
- 用户能完成基础跟读并获得简单反馈
- 用户能通过 HTTPS 地址在 iPhone 上打开并添加到主屏幕

## 10. 边界与限制
- 当前阶段不做手写训练
- 当前阶段不做复杂后台系统
- 当前阶段不做商业化能力
- 当前阶段的发音反馈仍属于基础版，不等同于专业纠音
